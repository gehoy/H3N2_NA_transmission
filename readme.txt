This folder contains the code for the H3N2 antibody transmission project (G Hoy, contact ghoy@umich.edu)

Code folders: 
01 code_primarymodel: core model presented in manuscript
02 code_additive_na: 03 code_antibody_transmission_working_additive: model with categorical high vs. low variable for each assay, with and without NA
03 code_additive_nat: model to test high vs. low NA in participants with detectable anti-HA immunity
04_code_sensitivity_analyses: Code for the sensitivity analyses
Simulation: code for generating the simulated datasets
analyze_MCMC_R_scripts: R scripts for analyzing and visualizing the MCMC results
